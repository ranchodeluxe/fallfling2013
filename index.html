Welcome to the Smackdown!

Let's learn how we can use Pierce County's awesome Python GDAL/OGR Cookbook. But first, let's install a bunch of junk on your computer if it's not already there.

Tooling, Installs, Data

Some of the tools we'll use today to explore the cookbook include:

  1. Shell # Windows people can use CMD, PowerShell or GitBash ( see below )
  2. Python
  3. GDAL/OGR Library and Python Bindings
  4. Mapfart.bom or QGIS # to visualize the input/outputs
  5. PostGIS # if you care about database support
  6. Shapefiles # i got mine from King County GIS Portal

We will not be walking people through installs. Installs depend on the operating system you are running among other factors. So here's a list of places to get started depending on your operating system:

Windows

There's no particular reason to use GitBash these examples. Using CMD or PowerShell in Windows is fine. Most of the time we will just be running python commands. But sometimes we might be running commands that look unfamiliar. If you want to explore a Linux-type shell environment than GitBash is a good place to start
